# 特色功能

## 只读|可写
只读场景下，涉及修改的按钮自动隐藏，界面更加简洁的同时避免误操作提高安全性。
在需要修改数据时，点击🔒解锁即可编辑。

## Redis信息
运行时间、键总数、客户端数量、内存使用量、持久化配置等关键信息重点展示。
所有参数表格展示，并添加每个参数的详细解释，便于用户快速识别和理解。

## Redis配置
- 参考：可以查看Redis8/7/6等默认配置参数
- 差异: 可以快速查看当前配置与默认配置的差异
- 说明：每个配置参数都有详细说明

## 终端命令
在集群模式下，支持指定特定节点执行命令以及自动广播命令（比如 config set xxx）。
支持命令的列表提示和详细解释

## 内存分析
允许精细化的配置每次扫描的个数，睡眠时间以及扫描总数等，便于快速内存占用大的键进行处理。
