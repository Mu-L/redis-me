# Redis管理客户端比较

[RedisME](https://www.hepengju.com)、[TinyRDM](https://redis.tinycraft.cc)、[AnotherRedisDesktopManager](https://goanother.com/cn)
三款都是非常优秀的开源免费跨平台的Redis管理客户端。**综合对比下，RedisME还是能打的**。

| 功能/特性 | RedisME               | TinyRDM             | AnotherRDM              |
|-------|-----------------------|---------------------|-------------------------|
| 项目启动  | 2025-06               | 2023-06             | 2019-01                 |
| Stars | 9                     | 12.6K               | 34.1K                   |
| 技术栈   | Tauri(WebView + Rust) | Wails(WebView + Go) | Electron(Chrome + Node) |
| 前端框架  | ElementPlus/Vue3      | NativeUI/Vue3       | ElementUI/Vue2          |
| 安装包   | 5~9M                  | 7~12M               | 59~83M                  |
| 启动速度  | 极快                    | 极快                  | 快                       |
| 内存占用  | 100M                  | 120M                | 230M                    |
| 界面美观  | 简洁美观                  | 中规中矩                | 略有逊色                    |
| 连接管理  | 完善                    | 最完善                 | 完善                      |              
| 只读模式  | 完美(动态隐藏)              | 不支持                 | 操作时提示                   |              
| 多连接实现 | 多窗口                   | 多标签                 | 手风琴                     |              
| 键展示   | 树形/列表(实时切换)           | 树形/列表               | 树形/列表                   |              
| 主信息   | 最丰富(详细解释)             | 丰富                  | 丰富                      |              
| 值显示   | 最美观                   | 美观                  | 一般                      |              
| 终端    | 完美(提示+手册)             | 较差(不支持中文输入)         | 一般                      |              
| 内存分析  | 完善(可定制参数)             | 不支持                 | 支持(不能定制参数)              |              
| 慢日志   | 完美支持                  | 支持(集群无法指定节点)        | 支持(缺少客户端展示)             |              
| 命令监控  | 完美支持                  | 支持(集群无法指定节点)        | 不支持                     |              
| 发布订阅  | 支持                    | 支持                  | 不支持                     |              
| 图表    | 丰富美观                  | 美观(点太少,不实用)         | 不支持                     |              
| 运行配置  | 完美支持(带解释+差异比对)        | 不支持                 | 不支持                     |              
| 客户端列表 | 完善列表                  | 支持(较简略)             | 不支持                     |
