- 优化: 原生Mutex锁改为parking_lot的Mutex，以便支持配置获取锁的超时时长，避免无限等待的情况
- 优化: 模拟数据插入时界面loading优化
- 优化: 升级所有依赖版本到最新版本，cargo fmt/clippy优化代码
- 缺陷: 键树形显示叶子节点和非叶子节点名称相同时的显示问题修复
- 缺陷: 批量删除键选择直接批量删除时卡死问题修复（加锁获取连接）

## v1.0.0 (2026-01-01)

- 🎉 迎接新年，第一个正式版本发布
- 升级: 支持自动更新、手动更新; 添加New标识和下载进度显示; 支持Gitee/Github双端更新
- 配置: 添加默认值展示，可选展示全部/差异，差异值特殊颜色展示，新增Redis8.4配置
- 主题: 支持浅色深色主题
- 字体: 支持自定义字体，支持浏览器字体获取和Rust读取系统字体两种方式，以便适应不同环境
- i18n: 支持中英文，支持英文的复数处理，信息和配置的选项详细解释支持中英文
