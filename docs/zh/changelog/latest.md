# 4.x 更新日志

## v4.1.0

- 键收藏功能
- 优化选择框样式
- 扫描暂停、继续

## v4.0.4 (2026-06-21)

- 命令日志弹框的多处细节优化
- 探测INFO命令权限，没有时不显示信息和图表界面
- 极简模式的判断优化

## v4.0.3 (2026-06-20)

- 命令日志加入异步执行的监控、发布/订阅和导入导出等
- 集群模式下INFO能力探测的缺陷修复
- 监控、发布/订阅的文字和图标优化
- 搜索框历史记录的高度优化
- 升级前后端依赖版本到最新

## v4.0.2 (2026-06-19)

- 键扫描整体重新设计: 实时显示 + 可停止扫描 #116
- 搜索框支持历史记录

## v4.0.1 (2026-06-18)

- 优化Scan时导致加载剩余所有键功能失效的修复
- 优化Scan的暂停时间100ms改为10ms

## [v4.0.0](https://mp.weixin.qq.com/s/U9DYq4LfoliE_eR1BKE5mg) (2026-06-18)

### ✨ 新增优化

- **命令日志**
  - 命令拦截、事件推送与可视化面板展示
  - 最近1s内的命令高亮颜色显示，便于追踪最新操作
- 键树: 新增 TYPE 缓存，避免展开时重复请求
- 键树: 选中行右侧添加快捷删除图标
- 搜索: 大数据量scan逻辑优化，避免界面卡死 #116
- 图标: 仅 macOS 保留安全区，Windows/Linux 无需安全区
- HTTL: HTTL命令是否支持的探测优化

### 🐞 修复问题

- 修复 `field_scan` 键不存在时误报类型不支持
- 删除键后不再触发 INFO 刷新，避免无权限用户总被弹框提示
