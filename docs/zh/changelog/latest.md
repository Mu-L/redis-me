# 3.x 更新日志

## v4.0.0 (2026-06-20)

### ✨ 新增优化

- **命令日志**
  - 支持命令拦截、事件推送与可视化面板展示
  - 支持 RESP3 错误识别、响应字段优化与前端展示优化
  - 弹框 UI 与交互体验优化
- **键树**
  - 新增 TYPE 缓存，避免展开时重复请求
  - 选中行右侧添加快捷删除图标
- **图标**: 仅 macOS 保留安全区，Windows/Linux 无需安全区

### 🐞 修复问题

- 修复 `field_scan` 键不存在时误报类型不支持
- 优化 TYPE 缓存：仅 bytes 入缓存，批量删与 FLUSHDB 整连接失效
