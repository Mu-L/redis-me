# 2.x 更新日志

## v2.7.0

### 🔗 连接管理
- 新增 SSH 隧道功能，支持密码和私钥两种认证方式
- 单机/集群连接使用不同图标，视觉区分更直观
- 连接选项复选框新增使用场景说明提示

### 🌍 国际化
- 后端错误提示全面支持多语言（中英文）

### 🎨 界面优化
- Set 类型简称调整为 E，避免与 String 的 S 冲突
- Stream 消费者组弹框增加图标并固定高度
- 信息刷新、键值删除/保存等交互细节样式优化

## [v2.6.0](https://mp.weixin.qq.com/s/rhs5S14vhNMHkNFmf5KONg) (2026-04-04)

- 键显示: **键类型的实时获取与图标显示**
- 键显示: **支持导入命令文件**
- 值显示: Stream的ID鼠标悬浮可以显示时间, 支持设置最大、最小id, 支持查询消费者组
- 慢日志: 集群模式支持指定节点, 默认排序改为时间倒序, 优化表格顺序与宽度
- 设置: **新增恢复默认值**, 部分设置添加提示
- 连接: 只读、集群、哨兵复选框添加提示

## [v2.5.0](https://mp.weixin.qq.com/s/dYXnLfvppK8HLAfU_RO32g) (2026-03-30)

- 键显示: **支持自定义数据库名称**
- 键显示: **新增批量删除键和清空数据库功能**
- 细节优化
  - **优化错误提示**: rust的错误和来源字符串一致时仅显示单个
  - 导出数据时的默认文件名添加服务版本信息: DUMP命令序列化值包含RDB版本，不兼容版本无法RESTORE导入
  - 连接DB索引不存在时自动切换到0: 单机数据库数量变少的特殊场景，可以正常连接
  - 更新日志链接的颜色，避免Mac系统深色主题下颜色过暗
  - 删除连接时的i18n支持

## [v2.4.0](https://mp.weixin.qq.com/s/rZYko6a9-T217pT8erSQ4w) (2026-03-27)

- 键显示: 整体设计优化
  - **新增复选框，多选后可批量导出键/删除键/更新键过期时间**
  - 优化平铺显示实现，树形显示支持按照键数量和字母排序并优化性能
  - 导出数据、导入数据、模拟数据从主菜单迁移到键显示扩展菜单
- 设置: 新增配置默认扫描键数量，默认键展示方式和默认树形排序方式
- 终端: **新增快捷键弹框提示**，优化命令表格弹框速度
- json格式化和解析优化
  - 值显示: **格式化支持非标json**（vscode同款）
  - 新增编辑: **支持json5格式的输入**（key可以不加引号等）
- 文档: 首页加入图标, 默认字体优化, 优化Apple故障排除
- 文档: 新增Redis客户端对比, 图表说明等使用手册

## [v2.3.0](https://mp.weixin.qq.com/s/WoZvSwyv-w4bxW0lzHh9OA) (2026-03-15)
