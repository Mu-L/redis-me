# 2.x 更新日志

## v3.0.0

- 数据编码: 支持UTF8, Hex, Binary, Base64格式
  - 新增键、重命名键
  - 值显示: 上方键、主区域值显示与保存、插入字段、编辑字段
- 安装包: 新增支持linux的arm架构 √
- 键显示: 数据库下拉框支持过滤并优化样式 √
- 慢日志: 服务器慢日志参数的快速调整
- 设置: 新增快速打开配置/应用/日志的目录按钮 √
- 路径解析: SSL/SSH等的文件路径支持包含家目录和环境变量 √

## [v2.9.0](https://mp.weixin.qq.com/s/WWc-iOfnxYtG5eiFRYCQQA) (2026-04-19)

### ✨ 新增功能

- 键显示：可配置显示**全称或简称**，高度可配置 #73
- 键显示：文件夹与键类型图标大小统一(14×14)，优化对齐方式 #75
- 值显示：集群模式下支持**查看键的哈希槽和所在节点**
- 配置弹框: **命令支持点击复制**，颜色和连接颜色保持一致
- 集群节点: **显示主从关系**(M1/M2/M3，S1/S2/S3)，显示主节点槽位
- 其他优化: 优化集群扫描速度和字段扫描速度

### 🐞 修复问题

- 集群模式: 清空当前库只在某个节点单独执行的问题
- 集群模式: HSCAN/SSCAN/ZSCAN扫描多个节点导致的字段重复问题
- 哈希字段级过期时间在多次扫描情况下的显示问题
