# 4.x 更新日志

## [v4.1.0](https://mp.weixin.qq.com/s/pM545fZPNiy3gxCvpDvmlw) (2026-06-27)

### ✨ 新增优化

- 新增**收藏键**功能
  - 右键收藏/取消收藏，已收藏键显示星标
  - 底部「我的收藏」入口，收藏模式仅显示当前连接/库的收藏键
  - 收藏列表支持关键词过滤；多选模式批量收藏/取消收藏
- **键区**
  - **支持历史记录**；下拉置于键列表下方，避免遮挡键列表
  - **实时显示扫描进度，支持暂停/继续扫描** #116
  - 精确查询使用Exists优化性能 #122
  - DB 选择器新增图标，样式优化
- **键值页**: 头部布局优化，TTL 显示在键名旁，操作收拢至收藏与更多菜单
- **命令日志**
  - 加入异步执行的监控、发布/订阅和导入导出等
  - 弹框交互与表格展示优化
- 其他细节优化
  - 探测INFO命令权限，没有时不显示信息和图表界面
  - 监控、发布/订阅的文字和图标优化
  - 极简模式判断优化
  - 平铺模式图标调整

### 🐞 修复问题

- 修复优化Scan时导致的「加载剩余所有键」失效

## [v4.0.0](https://mp.weixin.qq.com/s/U9DYq4LfoliE_eR1BKE5mg) (2026-06-18)

### ✨ 新增优化

- **命令日志**
  - 命令拦截、事件推送与可视化面板展示
  - 最近1s内的命令高亮颜色显示，便于追踪最新操作
- 键树: 新增 TYPE 缓存，避免展开时重复请求
- 键树: 选中行右侧添加快捷删除图标
- 搜索: 大数据量scan逻辑优化，避免界面卡死 #116
- 图标: 仅 macOS 保留安全区，Windows/Linux 无需安全区
- HTTL: HTTL命令是否支持的探测优化

### 🐞 修复问题

- 修复 `field_scan` 键不存在时误报类型不支持
- 删除键后不再触发 INFO 刷新，避免无权限用户总被弹框提示
